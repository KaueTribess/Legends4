[[Ação Padrão|<- Ação Padrão]]

### Descrição
Permite preparar uma [[Ação Padrão]] para ser realizada como uma [[Reação]] até o início do seu próximo [[Turno]], definindo um gatilho claro para ela.

Quando o gatilho acontecer, você pode usar sua [[Reação]] para realizar a ação preparada. Caso o gatilho não ocorra, a ação preparada é perdida.
