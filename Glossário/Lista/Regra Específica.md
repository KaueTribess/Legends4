### Descrição
Uma regra específica é uma regra que altera, limita ou substitui uma regra geral em uma situação determinada.

Quando uma regra geral entrar em conflito com uma regra específica, a regra específica prevalece apenas dentro do escopo que ela descreve.

### Características
- Prioridade: Regras específicas prevalecem sobre regras gerais quando houver conflito direto.
- Escopo: Uma regra específica só se aplica à situação, habilidade, magia, item, criatura ou efeito que ela menciona.
- Limite: Uma regra específica não deve ser expandida automaticamente para casos parecidos.
- Combinação: Regras específicas diferentes podem ampliar opções disponíveis, desde que cada uma respeite o próprio escopo.

### Exemplo
Se uma regra geral diz que uma ação exige uma [[Ação Padrão]], mas uma habilidade permite realizar essa mesma ação com uma [[Ação Bônus]], a habilidade prevalece naquele caso.
