### Descrição
É uma circunstância favorável que aumenta significativamente as chances de sucesso de uma [[Sistemas/Legends 4/Glossário/Lista/Rolagem|Rolagem]]. Quando uma [[Criatura]] tem Vantagem, ela joga dois dados iguais, em vez de um único, e pega o resultado mais alto dos dois como base para sua [[Sistemas/Legends 4/Glossário/Lista/Rolagem|Rolagem]]. Os modificadores são aplicados normalmente após a escolha do menor resultado. Não é possível acumular múltiplas Vantagens, se tiver Vantagem por duas razões diferentes, ainda rola apenas dois dados e escolhe o maior. Se uma [[Criatura]] tem tanto Vantagem quanto [[Desvantagem]] simultaneamente, elas se cancelam mutuamente, voltando a uma [[Sistemas/Legends 4/Glossário/Lista/Rolagem|Rolagem]] normal.
